Installation de GLPI

Extraire le contenu du fichier zip de glpi dans votre répertoire www ou htdocs dans notre cas www.
Procéder à l'Installation.

Si lors des tests d'extensions php certaines sont en erreurs, il vous faut les activer dans le fichier php.ini, pour cela in vous faut enlever le signe ";" devant les lignes correspondantes.

Créer une nouvelle base de données lors de l'installation

Installation du plugin fusion inventory

Extraire FusionInventory serveur dans le dossier plugin de GLPI
Activer FusionInventory dans la partie plugin sur l'interface web de GLPI

Connecter GLPI à votre AD

se rendre dans la partie configuration puis Authentification de GLPI
Clicker sur annuaire LDAP puis cliquer sur +
Remplir les informations comme ci-dessous en adaptant les champs

(&(objectClass=user)(objectCategory=person)(!(userAccountControl:1.2.840.113556.1.4.803:=2)))
