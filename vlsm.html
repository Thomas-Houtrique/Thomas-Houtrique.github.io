Créer les différents VLANS

enable
configure terminal
vlan *numéro vlan*
name *nom du vlan*
end

Faire un serveur VTP sur l'un des switchs

enable
configure terminal
vtp mode server
vtp domain *votredomaine*
vtp password *votre mot de passe*
vtp version 2

Faire un trunk entre les switchs et entre le routeur 0 et le switch principal

enable
configure terminal
interface *Votre interface*
switchport mode trunk

Mettre le reste des switchs en clients VTP

enable
configure terminal
interface *Votre interface*
switchport mode trunk

Autoriser les VLANS entre les switchs et les ordinateurs

enable
configure terminal
interface *Votre interface*
switchport access vlan *Numéro du VLAN*
